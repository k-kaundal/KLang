# KLang IDE Integrations

Official IDE and editor integrations for KLang.

## Available Integrations

### 1. Visual Studio Code ✅
Full-featured extension with LSP support.

### 2. IntelliJ IDEA 🚧
Full IDE support for JetBrains IDEs.

### 3. Sublime Text 🚧
Lightweight package for Sublime Text.

### 4. Vim/Neovim 🚧
Vim plugin with language server support.

### 5. Emacs 🚧
Emacs major mode for KLang.

See individual directories for detailed documentation.
