# KLang IntelliJ IDEA Plugin

Official IntelliJ IDEA plugin for KLang.

## Features

- Syntax highlighting
- Code completion
- Go to definition
- Find usages
- Refactoring support
- Debugger integration
- Run configurations
- Live templates

## Installation

### From JetBrains Marketplace (Coming Soon)

1. Open IntelliJ IDEA
2. Go to `Settings` → `Plugins`
3. Search for "KLang"
4. Click `Install`
5. Restart IDE

### Manual Installation

1. Download the plugin zip from [releases](https://github.com/k-kaundal/KLang/releases)
2. Open IntelliJ IDEA
3. Go to `Settings` → `Plugins`
4. Click the gear icon → `Install Plugin from Disk...`
5. Select the downloaded zip file
6. Restart IDE

## Configuration

### Run Configuration

1. Go to `Run` → `Edit Configurations...`
2. Click `+` → `KLang`
3. Configure:
   - **Name**: Your configuration name
   - **Script path**: Path to your .kl file
   - **Working directory**: Project root
   - **Program arguments**: Command line arguments
