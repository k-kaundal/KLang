# KLang Sublime Text Package

Official Sublime Text package for KLang.

## Installation

### Package Control (Recommended)

1. Open Sublime Text
2. Press `Ctrl+Shift+P` (Windows/Linux) or `Cmd+Shift+P` (Mac)
3. Type "Package Control: Install Package"
4. Search for "KLang"
5. Press Enter to install

## Features

- Syntax Highlighting
- Code Snippets
- Build System
- Auto-completion
