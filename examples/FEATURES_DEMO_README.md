# KLang v1.1-1.3 Features Example

This example demonstrates all the new features added in KLang v1.1-1.3.

## Features Demonstrated

1. **File System (fs.kl)**
   - File reading and writing
   - Path manipulation
   - File operations

2. **Date and Time (datetime.kl)**
   - Current date/time
   - Date formatting
   - Date arithmetic

3. **Collections (collections.kl)**
   - Stack
   - Queue
   - Set
   - OrderedMap

4. **Cryptography (crypto.kl)**
   - Hashing (SHA256)
   - UUID generation
   - Base64 encoding

5. **Testing (test.kl)**
   - Test suites
   - Assertions
   - Test runner

6. **Networking (net.kl)**
   - URL parsing
   - URL encoding/decoding
   - Query string handling

## Running the Example

```bash
klang run examples/feature_demo.kl
```

## Expected Output

The example will demonstrate each feature with output showing:
- File operations
- Date/time formatting
- Collection operations
- Cryptographic operations
- Test results
- URL parsing

## Notes

- This example uses all 6 new standard library modules
- Each section is clearly labeled
- The example cleans up after itself (deletes test files)
- All features work together seamlessly
